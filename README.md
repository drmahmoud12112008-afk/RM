## 💡 **عن المشروع**

**نظام RM** هو منصة متكاملة تهدف إلى توفير تجربة بحث ذكية مع إمكانية تخصيص الاختصارات والروابط. يتميز النظام بواجهة عربية أنيقة، ونظام مصادقة آمن باستخدام Firebase، وإمكانية تحويله إلى تطبيق سطح مكتب باستخدام Electron.

> "نصنع المستقبل، سطراً بسطر" - فريق RM
